## `$ whoami`

Soy **Luis Fernando Salguero Guzmán**, desarrollador de software con enfoque en tecnologías **Oracle** y desarrollo **Full Stack**. Actualmente finalizando la Licenciatura en Ingeniería en Sistemas de Computación en la **Universidad Fidélitas**, Costa Rica.

Me muevo cómodo entre escribir PL/SQL para sistemas empresariales y levantar un stack completo con React, Node.js y Docker en Oracle Cloud. Si hay un problema técnico, voy a encontrar la forma de resolverlo — preferiblemente de la forma más eficiente posible.

Trabajo con **CRC Coding S.A.**, empresa costarricense especializada en soluciones Oracle, donde desarrollo y mantengo aplicaciones de gestión empresarial en APEX.
